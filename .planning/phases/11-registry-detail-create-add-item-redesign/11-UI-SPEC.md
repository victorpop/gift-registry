**Special: Occasion tile glyphs**

Glyphs (тМВ тЩб тЧР тЬж тЭЕ +) render at `typography.displayM` (`InstrumentSerifFamily`, 22 sp, `FontStyle.Italic`). Color: `colors.accent` when unselected, `colors.accentInk` when tile selected. If a glyph lacks Instrument Serif coverage at 22 sp (verify at preview time), fall back to the nearest unicode equivalent (e.g., `\uD83C\uDFE0` for тМВ if serif italic fails).

**Special: Share banner URL**

The share URL uses `typography.monoCaps` as the font family base but renders at body-equivalent size with `FontWeight.SemiBold` (W600) and `colors.accent` fill. This is the one monoCaps variant with non-standard weight. Implement as:
```kotlin
Text(
    text = shareUrl,
    style = GiftMaisonTheme.typography.monoCaps.copy(
        fontWeight = FontWeight.SemiBold,
        color = GiftMaisonTheme.colors.accent,
    )
)
```

**Special: Create registry headline italic-accent construction**

"What's the **occasion?**" headline is NOT a single Text. "What's the" renders in `typography.displayS`, `FontStyle.Normal`, `colors.ink`. "occasion?" renders in `typography.displayS`, `FontStyle.Italic`, `colors.accent`. Implement as two separate `Text` composables in a `Column` or as `buildAnnotatedString`.
