
   ```bash
   firebase login
   ```

3. Create the Firebase project (choose ONE):
   - **Option A (CLI):** `firebase projects:create gift-registry-ro --display-name "Gift Registry"`
   - **Option B (Console):** Go to https://console.firebase.google.com -> Add project -> name it `gift-registry-ro`

4. In the Firebase Console (https://console.firebase.google.com/project/gift-registry-ro):
   - Enable **Authentication** (Authentication -> Get started -> enable Email/Password provider)
   - Create **Cloud Firestore** database (Firestore Database -> Create database -> select `europe-west3 (Frankfurt)` region -> Start in production mode)
   - Enable **App Check** (App Check -> Register app -> skip enforcement for now)

**CRITICAL:** Firestore region `europe-west3` cannot be changed after creation. This is permanent.

5. Report back the Firebase project ID (e.g., `gift-registry-ro` or whatever was chosen).
