## Decisions Made

- **FabSlot column mirrors NavItemSlot exactly (minus the clickable, which lives on the FAB itself)** — keeps label-baseline alignment a property of the shared column layout pattern rather than a hand-tuned offset, so future slot tweaks (e.g. changing the 4 dp icon-to-label gap) only need to be made in one place to remain consistent.
- **Inner 44 dp Box has no background/clip** — it is purely an invisible alignment scaffold. Adding a pill background would visually compete with the FAB and is not part of the handoff design.
- **`requiredSize` ordering:** `requiredSize(54.dp)` before `offset(y = (-22).dp)` so the size constraint resolves first and the visual lift is applied to the already-sized 54 dp circle. Reversing the order would still work geometrically but reads less naturally and would mask the intent.
